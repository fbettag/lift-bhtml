ag.bett.lift.bhtml

BHtml

object BHtml extends AnyRef

Adds support for Twitter Bootstrap CSS-classes for validation, also makes validation for all KeyedMapper-derived Models easy through a Twitter BootstrapJS Popover.

All selectors are jQuery Selectors.

The save-parameter makes fields auto-commit/auto-save the record on Event (blur).
